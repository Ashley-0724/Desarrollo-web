Quiénes Somos

El proyecto "Aprendizaje y Desarrollo de Aplicaciones Web" tiene como objetivo fortalecer los conocimientos sobre el diseño y creación de sitios web utilizando HTML5, CSS y Bootstrap. Durante el desarrollo del proyecto se aplican etiquetas semánticas, estructuras organizadas y elementos multimedia para construir páginas modernas, funcionales y fáciles de navegar.

Además, se busca comprender la importancia de una correcta organización del contenido, mejorar las habilidades de programación y fomentar la creatividad en el desarrollo de aplicaciones web.
